Jika API Kamu di Belakang Akamai (Reverse Proxy)
Yang perlu dikonfigurasi di appsettings.json bukan AllowedHosts, tapi Forwarded Headers:
​

csharp
// Program.cs
app.UseForwardedHeaders(new ForwardedHeadersOptions
{
    ForwardedHeaders = ForwardedHeaders.XForwardedFor | ForwardedHeaders.XForwardedProto,
    KnownNetworks = { new IPNetwork(IPAddress.Parse("x.x.x.x"), 24) } // IP Akamai edge
});
